Welcome to Daniel & Nadav's Open Weather project.

The program was written as a final project at programming-languages course.

How to use the API: You can set a new Location object and set it's city attribute to any city you like. Initiate a WeatherDataServiceFactory instance and then send to it's GetWeatherDataService method the Location object. The API returns a parsed Json object and you can choose what output you want to print.

Example: WeatherDataServiceFactory obj = WeatherDataServiceFactory.Instance; 

        Location location = new Location();                                                    
        location.city = "London";

        var tmp = obj.GetWeatherDataService(location);
        System.Console.WriteLine("Country: " + tmp.Sys.Country.ToString() + "\nCity: " + tmp.Name.ToString() + "\nTempreture: " + tmp.Main.Temp.ToString() + " (Kelvin)" + "\nWind Speed: " + tmp.Wind.Speed.ToString() + "\nPressure: " + tmp.Main.Pressure.ToString() + " (Kelvin)" + "\nHumidity: " + tmp.Main.Humidity.ToString());

Enjoy.
